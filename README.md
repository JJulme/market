# market

A new Flutter project.

Flutter를 이용해서 간단하게 만든 게시판 앱

## 주요기능

JSON Placeholder를 이용해서 REST API 데이터를 받아온다.

<https://jsonplaceholder.typicode.com/>

게시판의 게시물을 가져오고 해당 게시물의 내용과 댓글을 불러온다.

사용자의 정보를 조회할 수 있다.
